# Contributing to Finomaly

Thank you for considering contributing to Finomaly! We welcome all kinds of contributions, including bug reports, feature requests, documentation improvements, and code enhancements.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Open an issue** to discuss your proposed change before making a big change.
3. **Write clear, concise commit messages**.
4. **Create a pull request** with a clear description of your changes.
5. **Ensure your code passes all tests and linting.**

## Code of Conduct
Please be respectful and inclusive in all interactions.

## Need help?
Open an issue or start a discussion on GitHub!
