# Security Policy

## Supported Versions

The following table lists which versions of this project are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing iletisim@barisaksel.com.  
You will receive a response within 7 days.  
All reports will be reviewed and addressed promptly.  
Please do not disclose security issues publicly until they have been resolved.
